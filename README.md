# cpp-solitaire
Ye Olde solitaire (klondike) in C++

## Compiling
### Prerequisites
- cmake >= 3.11.2
- make
- g++ (or something comparable that compiles c++11)

### Instructions
- Clone repository and move to `/src/`
- Run `cmake .`
- Run `make`
- Enjoy
